# IFSE
The IFSE scheme is a hybrid numerical stiff integrator for solving stiff reaction-diffusion problems arising in marine and ecological systems. The numerical scheme, in its development, combines Strang operator splitting, SSP-RK2 time-stepping, and a dimensional splitting method, which improves efficiency for multi-dimensional coupled reaction-diffusion systems. The MATLAB codes for running some 2D model test problems used to validate the accuracy, convergence, and robustness of the proposed scheme are available in this repository. We provide separate README files in code zip folders to explain how to reproduce the results reported for all the test problems.
